# Proyecto Nodejs + Handlebars

## Instalación

Se instalaran todos los paquetes necesarios para su funcionamiento

```node
npm install
```
## Ver en desarrollo

Se activará el modulo de Nodemon para ver su correcto funcionamiento en modo Desarrollo

```node
npm run dev
```
